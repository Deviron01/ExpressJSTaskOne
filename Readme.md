**Task One Express js : Creating a Personal Profile API**
--

***Objective***
- Build a simple API that return information about yourself

**Requirements**
-
***1.Basic set up***
- Create a new folder called `my-profile-api`
- Initialize npm `npm innit -y`
- Install Express `npm i express`
- Create a `server.js`

***2.Create a basic route***

|Route|What it should return|
|--------------|--------------|
|/|"Welcome to my Profile API"|
|/name|Your name as a text |
|/age|Your age as a text|
|/course|Your course|
|/bio|A short bio about yourself as a text|

***3.What to do***
-
- Run the server with `node server.js`
- Open the browser and go to http://localhost:3000
- Test all routes
- Try changing the response and see whether the server update
